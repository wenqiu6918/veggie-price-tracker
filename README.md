# 菜价本 - 菜价记录和比价工具

一个简单易用的移动端菜价记录应用，帮助你记录和比较不同地点的菜价。

## 🚀 快速开始

直接在浏览器中打开：
```
https://raw.githubusercontent.com/wenqiu6918/veggie-price-tracker/main/index.html
```

或者访问：
```
https://cdn.jsdelivr.net/gh/wenqiu6918/veggie-price-tracker@main/index.html
```

## ✨ 功能特性

- 📝 记录菜价（名称、价格、地点、日期）
- 📷 拍照识别（支持OCR）
- 📊 智能比价（自动对比同种蔬菜价格）
- 💾 本地存储（数据保存在浏览器）
- 📥 批量导入（支持文本和CSV）
- 🎨 浅蓝色主题

## 📱 使用方法

1. 点击"记录菜价"按钮
2. 填写蔬菜信息
3. 保存后自动更新列表
4. 切换到"比价"页面查看价格对比

## 💡 提示

- 所有数据保存在浏览器本地，不会上传到服务器
- 清除浏览器缓存会丢失数据，建议定期导出备份
- 支持在手机浏览器中使用
